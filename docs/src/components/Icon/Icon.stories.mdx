# Icon

> [Icon] component contains the vector icons that we've designed.

## Usage

```tsx
import {Icon} from 'dooboo-ui';

<Icon {...props} />;
```

## Props

|        | Types      | Description                                                             |
| ------ | ---------- | ----------------------------------------------------------------------- |
| name?  | `IconName` | Name of the `dooboo-ui` icons. They are typed when you use `typescript` |
| size?  | `number`   |                                                                         |
| color? | `string`   |                                                                         |
